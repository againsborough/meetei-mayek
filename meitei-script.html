For use in HTML, the Meetei script can be simply be copy-pasted, like this: ꯃꯤꯇꯩ ꯃꯌꯦꯛ, which works great when you already have a Meetei Mayek keyboard or have the script from somewhere else.

Otherwise, you can use the Unicode. Use the usual syntax for entering code points in HTML. For instance, you can write ꯀ.
